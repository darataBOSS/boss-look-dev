# 深海の観測ルーム セットアップ手順（VR）

「ガラスの小部屋の中（最初/最後）＋外は深海（メイン）」で、**動くアバターが室内↔外を行き来する** VR シーンの作り方。**ルックの切り替えコードは不要**で、すべて1回のベイク＋プローブで成立します。

## 考え方
- **空間で違う光（室内=暖色 / 外=青）はベイクとプローブが担当**。アバター（動く）は**位置に応じてライトプローブが自動補間**するので、ドアを抜けると自然に色が変わる（ランタイム処理なし）。
- **HDRI（スカイボックス）は外の深海1つだけ**。室内は普通のライト＋ベイクで作る（室内に2つ目のHDRIは置けない/不要）。
- **フォグ・カラーグレードは画面全体**に効く（場所で分けられない）。深海の濁りは「ほどほどのグローバルフォグ（案B）」で出す。室内も少し青むが、深海の部屋として自然。

## 手順
1. **レイヤー準備**: アバターと魚を専用レイヤー（例: `Dynamic`）に置く。
2. **Look 作成**（BOSS > Look Dev → ＋新規）。Target=SelfApp / Context=VR。
3. **ライティング（土台）**
   - **HDRI** に深海HDRIを設定（外の環境光・反射・ガラス越しの背景）。
   - **ライトリグ=太陽**（上から青）＝海の主光。**室内ライト（暖色）は自分で配置**（ツールはまとめてベイクする）。
   - **Static モード = ExcludeLayers** にして **動的レイヤーに `Dynamic` を指定**（アバター/魚は非Staticのまま＝プローブで光る）。
   - **メイン範囲（室外/海）**にライトプローブ＋リフレクション生成（魚用）。
   - **「室内ゾーンを使う」ON** → 室内範囲を部屋に合わせて、室内ライトプローブ＋室内リフレクション生成。
4. **背景**: VR=スカイボックス表示（深海HDRIが見える）。**フォグ=案B（ほどほどの青）**。
5. **カラー**: わずかに寒色。必要ならマテリアル底上げ。
6. **ガラス**: 観測窓のモデルとシェーダは自作（中間くらいの tint＋フレネル、透明度高めで深海が見える）。
7. **🔥 ベイク**。

## 結果
- アバター：室内では暖色、外では青に**自動**（位置でプローブ補間）。
- 反射：室内オブジェクトは部屋を、外は海を映す（室内/室外リフレクションプローブ）。
- 魚：外の青い光で正しく照らされる（外のライトプローブ）。
- 切り替えコードなし・1回のベイクで完成。

## 注意
- **アバター/魚を Static にしない**（Staticにするとプローブを使わず固定の光になる）。Static は ExcludeLayers で除外。
- フォグはグローバルなので室内も薄く青む（深海の部屋として自然。強すぎたら濃度を下げる）。
- 最終確認は**実機（Pico）**で。
